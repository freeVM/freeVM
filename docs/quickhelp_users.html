Getting Started For Users

The following instructions are for users that wish to use Apache Harmony for running their Java programs. These steps require no development tools our need to compile source code.

Any required dependencies are listed below per platform.

Download the Apache Harmony Snapshot

The first step is to download the binary distribution appropriate to your platform. We currently distribute binaries for Linux and Microsoft Windows. Please follow the instructions on the download page to choose and download the software.

Currently, the Apache Harmony project has no formal releases of its software. Instead, the project is offering pre-built snapshots that are expected to run well on the paltforms offered.

Unpack the binary distribution

Once you have optained the appropriate binary distribution, you need to unpack it on your machine. In all cases, the distribution will unpack into a jre directory. We shall refer to this directory as the 'root directory' of the distribution.

Linux Users

If you are running Linux, you will need to use the tar program as follows : 

$ tar -tzf latest-harmony-jre-linux.tar.gz

This will unpack the JRE directory tree into the current directory.

Windows Users

If you are running Windows, you will need to use one of the many zip utilities available for windows to unpack the distribution. Here are three popular ones, but anything that can handle a zip file will work :

  • Info-ZIP (open source software)
  • 7-Zip (free software)
  • Winzip (commercial)

Using the utility, extract the distribution zip file. this will create the JRE directory tree.

Configuring Your Environment

Apache Harmony doesn't need any further environmental setup, but sometimes other programs do. Also, it's convenient to have the Apache Harmony executables on your path.

Generally speaking, you'll want to set your JAVA_HOME and PATH environment variables. Exactly how to do this depends on the operating system and/or shell you are using.

Linux Users

The following works for the Bash shell. 

$ cd jre
$ export JAVA_HOME=`pwd`
$ cd bin
$ export PATH=`pwd`:$PATH

Note that this assumed your current directory was the parent directory of the Harmony JRE. Also, please note those are 'backtick' characters, not apostrophes.

Windows Users

The following example assumes you unpacked the zip file into the root directory "C:\". Please adjust as necessary. 

C:> cd jre
C:\jre>set JAVA_HOME=C:\jre
C:\jre> cd bin
C:\jre\bin> set PATH=C:\jre\bin;%PATH%

Using Apache Harmony

Now you should be ready to use Apache Harmony. Note that we don't yet include javac, so continue to use the compiler from some other JRE. You should be able to run programs normally. 

$ java Foo
$ java -jar myjar.jar
