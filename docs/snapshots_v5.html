Snapshot Builds and Test Results for Harmony 5.0

We offer the following Harmony binaries for the Linux and Windows platforms:

  • The Java Runtime Environment (JRE) lets you run your Java programs.
  • The Java Development Kit (JDK) lets you build and run your Java programs. It includes javac and other standard JDK tools.
  • The Harmony Development Kit (HDK) is targeted at Harmony developers. It includes the pre-built class library and VM binaries, necessary include files, and dependencies. It is not a replacement for the common JDK. We plan to include the Harmony test suite in the HDK in future. Please see HARMONY-950 and HARMONY-984 for details and current progress.

These are not official releases of the Apache Software Foundation, nor are they complete or compatible with the Java specification. They are snapshots provided to make testing and experimentation easier. Results of automatic testing are available.

Note

We recommend that you set JAVA_HOME to the root directory of the distribution, and put the bin/ directory on your PATH.
