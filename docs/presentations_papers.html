Presentations

From the Apache Harmony presentations listed below, you will get information about background, motivation, history, status and future of the Apache Harmony project, learn how Apache is building a compliant independent implementation of Java SE, and find out interesting facts about developing the Java Platform with Eclipse.

These presentations might inspire you to participate in Harmony community activities.

  • 2007
    • Apache Harmony: Implementing Java SE in Open Source, by Geir Magnusson Jr., presented at Emerging Technologies for the Enterprise Conference 2007.
    • EclipseCon 2007 Short Talk, by Tim Ellison, March 2007.
    • EclipseCon 2007 Poster, by Tim Ellison, March 2007.
  • 2006
    • Apache Harmony Approach to Implementing Java SE Platform, by Tim Ellison, Geir Magnusson Jr., 2006 JavaOne.
  • 2005
    • Apache Harmony, by Geir Magnusson Jr., 2005 JavaOne.

Research Papers

The papers listed below, provide a deep insight into the Apache Harmony research work.

  • Hardware Atomicity for Reliable Software Speculation*, by Naveen Neelakantam, Ravi Rajwar, Suresh Srinivas, Uma Srinivasan and Craig Zilles, presented at 34-th International Symposium on Computer Architecture, San Diego, CA, June 2007.
  • Task-pushing: a Scalable Parallel GC Marking Algorithm without Synchronization Operations*, by Ming Wu, Xiao-Feng Li, presented at 21-st IEEE International Parallel and Distributed Processing Symposium, Long Beach, CA, March 26, 2007.
  • Behavior Characterization and Performance Study on Compacting Garbage Collectors with Apache Harmony *, by Chunrong Lai, Ivan T Volosyuk, and Xiao-Feng Li, presented at Tenth Workshop on Computer Architecture Evaluation using Commercial Workloads held with HPCA-13, Phoenix, AZ, Feburary 2007.
