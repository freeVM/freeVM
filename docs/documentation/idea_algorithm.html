IDEA Algorithm

The Bouncy Castle Crypto API jar, bcprov.jar, contains an implementation of the patented IDEA algorithm. For details of the restrictions on this algorithm, see the Bouncy Castle documentation. Harmony contains a version of the bcprov.jar with the algorithm removed. The original jar file can be obtained from the Bouncy Castle release page.
