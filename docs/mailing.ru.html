Список рассылок

В настоящее время доступны четыре адреса электронной рассылки проекта Harmony (в том числе в публичных архивах):

dev@harmony.apache.org
Основная рассылка для обсуждения изменений и развития проекта. Подписка на эту рассылку доступна всем, но только подписчики могут помещать сообщения в рассылку.
commits@harmony.apache.org
Все проекты сообщества Apache хранятся на репозиториях общего доступа с использованием контроля версий SVN. Только некоторые разработчики, участвующие в проекте (члены сообщества, committers), имеют права на запись в этих репозиториях; остальные имеют права на чтение через анонимный доступ в SVN.
alerts@harmony.apache.org
Рассылка автоматических оповещений от различных систем построения и тестирования кода, используемых сообществом Harmony.
private@harmony.apache.org
Приватная рассылка членов комитета по управлению проектом используется для обсуждения тем, не подходящих для публичного обсуждения, например, вопросов юридического, личного характера или связанных с безопасностью проекта, до публикации их решения. Подписка на эту рассылку доступна только членам комитета.

Отметим, что для переписки используется только английский язык. Для сообщения об ошибках или пожеланиях на русском, воспользуйтесь нашей системой учета дефектов.

Рассылка для разработчиков

Данная рассылка используется разработчиками проекта для обсуждения планов, принятия решений, голосования по техническим вопросам, и т.д. Эта рассылка достаточно объемна, хотя часть обсуждений также проходит в канале IRC .

  • Подписаться на dev@harmony.apache.org
  • Отписаться от dev@harmony.apache.org
  • Просмотреть архивы harmony-dev@incubator.apache.org (май 2005 - октябрь 2006)
  • Просмотреть архивы dev@harmony.apache.org (с ноября 2006)

Примечание

В настоящее время, для формирования единого сообщества в рассылку включается вся переписка разработчиков - обсуждение Java библиотек классов, виртуальной машины, документации, поддержки сайта, разработки приложений и т.п. Предполагается, что позднее рассылка будет подразделена на несколько обсуждений различной тематики.

Для облегчения сортировка писем рассылки, мы просим добавлять следующие маркеры к теме Вашего сообщения:

  • [general] - общие вопросы
  • [announce] - объявления
  • [legal] - юридические вопросы
  • [classlib][module] - вопросы по библиотекам классов (по желанию с указанием названия модуля)
  • [drlvm][module] - вопросы по виртуальной машине DRLVM (по желанию с указанием названия модуля)
  • [jchevm][module] - вопросы по виртуальной машине jchevm (по желанию с указанием названия модуля)
  • [bootvm][module] - вопросы по виртуальной машине bootvm (по желанию с указанием названия модуля)
  • [doc] - документация
  • [tools] - программы
  • [testing] или [qa] - тестирование

Просто вставьте соответствующий маркер перед темой сообщения.

Пример

To: dev@harmony.apache.org
From: harmony_user@openjava.org
Subject: [classlib][io] Problem with java.io.OutputSocket
....

Рассылка системы контроля исходного кода

Подписавшись на рассылку, Вы сможете получать уведомления об всех изменениях в исходном коде Harmony на репозитории (со списком изменений), а также об изменениях на Вики-страницах проекта и изменениях в системе учета дефектов.

  • Подписаться на commits@harmony.apache.org
  • Отписаться от commits@harmony.apache.org
  • Просмотреть архивы harmony-commits@incubator.apache.org (май 2005 - октябрь 2006)
  • Просмотреть архивы commits@harmony.apache.org (с ноября 2006)

Рассылка оповещений

Подписавшись на эту рассылку, Вы сможете получать оповещения от различных систем построения и тестирования кода, используемых сообществом Harmony с информацией о неисправных сборках кода и результатах тестирования.

  • Подписаться на alerts@harmony.apache.org
  • Отписаться от alerts@harmony.apache.org
  • Просмотреть архив alerts@harmony.apache.org

Другие рассылки

Комитет по управлению проектом Harmony также использует приватную рассылку для решения административных вопросов. Подписка на рассылку доступна только членам комитета, куда входит большинство членов сообщества.
