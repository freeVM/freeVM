Virtual Machines that Work with Harmony

Besides our own DRLVM virtual machine, there are also other virtual machines that work with the Apache Harmony class library:

  • IBM J9: IBM has made their production-grade virtual machine available for project use under a non-open source, evaluation binary license. This VM works with the Harmony class library.
  • BEA JRockit: BEA has made their production-grade virtual machine available for project use under a non-open source, evaluation binary license. This VM works with the Harmony class library.
